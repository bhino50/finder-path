Install

Download the DMG, drag to Applications, approve Finder access.

This build is locally signed. If macOS blocks the first launch, right-click FinderPath and choose Open. The Finder Automation prompt appears the first time the app reads the current Finder path. Terminal permission appears when using Terminal-backed agent launch actions. Ghostty and agent buttons are optional and can be hidden or pointed at a custom executable path in Settings.
