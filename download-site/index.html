Install

Install the signed macOS app or the Linux sibling.

On macOS, download the DMG and drag FinderPath to Applications. FinderPath is signed with a Developer ID and notarized by Apple, so the current release opens normally after installation.

If macOS blocks an older or locally built copy, delete it and download the current release again. The Finder Automation prompt appears the first time the app reads the current Finder path, and Terminal permission appears when using Terminal-backed agent launch actions. On Linux, download the separate finderpath-linux installer to add the finderpath-linux command, Nautilus scripts, a Dolphin service menu, supported desktop dependencies, and an optional GTK/AppIndicator tray launcher.
