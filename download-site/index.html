Install

Install the macOS app or the Linux sibling.

On macOS, download the DMG and drag FinderPath to Applications. FinderPath is open source and not Apple-notarized, so macOS blocks the very first launch — this is expected and you only need to bypass it once:

  • macOS 13–14 (Ventura / Sonoma): right-click (or Control-click) FinderPath in Applications, choose Open, then click Open in the dialog.
  • macOS 15+ (Sequoia / Tahoe): double-click FinderPath once (it gets blocked), then open System Settings → Privacy & Security, scroll to the "FinderPath was blocked…" message, click Open Anyway, and confirm with Open.

After that, the Finder Automation prompt appears the first time the app reads the current Finder path, and Terminal permission appears when using Terminal-backed agent launch actions. Prefer not to bypass Gatekeeper? Build it from source — it is a single Swift file. On Linux, download the separate finderpath-linux installer to add the finderpath-linux command, Nautilus scripts, a Dolphin service menu, supported desktop dependencies, and an optional GTK/AppIndicator tray launcher.
